# Security Policy

This repository is a portfolio project and does not process live customer, revenue, or campaign data.

## Reporting

If you identify a security issue in the development workflow or project shell, report it through GitHub Security Advisories or through the official contact paths listed on [Kinetic Gain](https://kineticgain.com/).

## Scope

- Demo data remains synthetic.
- No production credentials or analytics secrets should be committed.
- Package hygiene should continue through Dependabot and CI.
